О проекте

Этот сайт создан для вязальщиц, которые ищут вдохновение и новые идеи. Здесь собраны бесплатные схемы и описания вязаных изделий. Нажимайте на изображения или названия, чтобы открыть полную схему.

Все схемы предоставлены для личного использования. При публикации своих работ указывайте источник.

Основные узоры

Популярные узоры для вязания спицами:

  1. Лицевая гладь — самый простой базовый узор.
  2. Изнаночная гладь — обратная сторона лицевой.
  3. Резинка 1x1, 2x2 — для манжет и планок.
  4. Платочная вязка — все петли лицевые.
  5. Ажурные узоры — с накидами и убавлениями.
  6. Косы и жгуты — рельефные узоры.

Для каждого узора есть схема:

  • Схема лицевой глади
  • Схемы резинок
  • Схемы кос

Советы по вязанию

Как подобрать пряжу:

  • Для свитеров — шерсть или полушерсть
  • Для летних вещей — хлопок или лен
  • Для шапок — теплая объемная пряжа

Основные правила:

  1. Всегда вяжите образец 10x10 см
  2. Стирайте образец, чтобы узнать усадку
  3. Подбирайте спицы на 0.5-1 мм толще пряжи
  4. Записывайте все изменения в схеме

Условные обозначения в схемах:

            | - лицевая петля
            - - изнаночная петля
            O - накид
            / - 2 петли вместе с наклоном вправо
            \ - 2 петли вместе с наклоном влево
